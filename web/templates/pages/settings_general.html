Operational Notes

  • 1. Point the DNS A record for TALOS_DOMAIN to this server before switching into domain mode.
  • 2. In external proxy mode, the shared Traefik edge proxy must watch {{.Current.EdgeNetwork}} and route both the Talos container labels and any app container labels created by new deploys.
  • 3. When an app in external mode uses a custom domain, Talos performs a brief stop/start cutover during deploy so the shared Traefik router points at only one live container at a time.
